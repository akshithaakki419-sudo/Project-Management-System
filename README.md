# Project Management System

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) application for managing projects and tasks.

## Features
- User Authentication
- Create and Manage Projects
- Create, Update and Delete Tasks
- Dashboard for Project Tracking
- Responsive User Interface

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Version Control: Git & GitHub

## Project Structure
```
project-management/
│
├── frontend/
├── backend/
├── README.md
└── package.json
```

## Installation

### Clone the Repository
```bash
git clone https://github.com/akshithaakki419-sudo/Project-Management-System.git
cd Project-Management-System
```

### Install Dependencies

Frontend:
```bash
cd frontend
npm install
```

Backend:
```bash
cd backend
npm install
```

### Run the Project

Backend:
```bash
npm start
```

Frontend:
```bash
npm start
```

## Environment Variables

Create a `.env` file in the backend folder and add:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
